AutoRigger - Human

 

Auto Rigger-Human allows users to freely generates the necessary joints and automatically creates the rig. (github link)

 

Features in 1.0 version (12/29/2018):

 

- Adjustable spine & finger numbers

- Secondary reverse foot control and forearm twist

- One-click add controller & bind skin

 

Update 1.1(2/25/2019)

 

-Supports Non-Tpose/Tpose Character Model

-Include IK/FK switch for Character arm

-Include Foot Bank attribute

 

Update 1.2(3/24/2019)

 

-Auto Facial Rig Feature added

-Eye, Brow, Lip and Jaw control support

-Advance Spine Control added

 

Poker Spring Rig (WIP)

 

Poker Spring Rig allows you to procedurally generate a controllable deck along a curve to create poker spring effects. It's also fully keyable.

 

Features:

 

- Double-sided texture and shuffle function

- Flexible card numbers and three tangent type

- Default poker bending and random rotation offset

- Controllable path animation

AutoRigger - Vehicle (WIP)

 

Auto Rigger - Vehicle procedurally generates wheel and body rig for vehicles.

 

Features:

 

- Adjustable rig length and width

- Physics based steering and driving

- Flexible control of wheel and body rotation

 

 

 

Scene Manager

 

Scene Manager allows users to manage their scene flexibly. By using simple instructions, users can build their scene with props that they created.

 

Features:

 

- Scene information preview

- Auto screenshot

- Three different sort methods

- Easy import and save

 

Motion Coverter

 

Motion Converter allow users to convert root motion into in place motion.

 

Features:

 

- Add root joint (ideal for game engine import)

- Set cycle range

- Options forward direction

Easy Deformer

 

Easy Deformer creates light-weighted and fast deformer to the character.

 

Features:

 

- Non-linear bend deformer on all direction

- Adjustable influence weight

- Stretch and Squash

 

Gear Creator

 

Gear Creator is a simple script that allows user to create and modify different kinds of gears easily.

 

Features:

- Five parameters gear creation

- Real-time update gear adjustment
